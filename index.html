Добавление информации к файлам

Чем точнее Вы будете указывать информацию,тем легче модераторам будет понять что Вы загрузили и файлы быстрее попадут в базу. Обязательное заполнение полей информации при добавлении файлов:

  • Для сериалов/фильмов информация берётся с KinoPoisk ID, IMDb ID только русское название
  • Для аниме основной источник информации world-art.ru, если нет нужного релиза дополнительный источник информации Shikimori ID
  • При добавлении дорам обязательно нужно указывать MyDramaList ID
  • Название на русском
  • Год
  • Номер сезона и номер серии
  • Озвучка как указана в базе
  • KinoPoisk ID
  • IMDb ID
  • Shikimori ID для аниме
  • Ссылка на MyDramaList ID для дорам
  • Ссылка на world-art.ru для аниме
  • Ссылка на торрент
Комментарии указываются если есть какие-то нюансы с файлом (доп. информация о озвучке, удалить файл и т.д.)
Поле "Начать с секунды" можно оставлять пустым, оно создано только для того, если Вам нужно указать пропуск в серии.
Если Вы загружаете новинку, можно выставлять "высокий" приоритет, чтобы конвертер быстрее брал такие файлы на обработку.

Общая информация

После окончания конвертации файла, слева от него будет кружок, показывающий, добавлен ли ваш файл в базу модератором или еще нет. Если вам хочется как можно скорее добавить плеер на сайт, то можете скопировать ссылку на плеер прямо из конвертера.

Копирование полей

Для автозаполнения полей, Вы можете копировать поля информации в базе/конвертере и вставлять их при добавлении серии в конвертере.

Почему мой файл удалён?

Ваш файл может попасть на удаление если:

  • Вы заполнили не все поля информации, при условии что в базе есть вся информация.
  • Допущена ошибка в информации.
  • Нерпавильный источник информации.
  • Загуржены отдельные серии в сезоне, мы не добавляем незаконченные релизы.
  • Если загрузили контент содержащий откровенные сцены (+18).

Как вернуть файлы с удаления?

Для того чтобы вернуть файлы с удаления нужно:

  1. Написать в чат конвертера
  2. Загрузить заново файл с правильной информацией если не хотите ждать ответа тех. поддержки
